# To-Do App

A simple To-Do application built with React and Redux.

## Features

- Add, edit, delete tasks
- Mark tasks as completed
- Persistent storage using local storage

## Technologies Used

- React
- Redux
- React Bootstrap
- React Bootstrap Icons

## Getting Started
